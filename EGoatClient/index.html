About eGoat

eGoat P2P Client is a peer-to-peer client compatible with eGoat P2P Server made as a part of an academic assignment. It's a product of collaboration between Kamila Korzec and Jakub Pietkiewicz.

Source code for both server and client is available down below via links to public repositories on GitHub.

eGoat Client

eGoat P2P Client is built as a single page application and is based on jQuery and WebSocket. User can upload (share) files from directory and search and directly download files shared by other users.
